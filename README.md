# subir_archivo_temporalmente

## Uso
```text
Utilidad para subir archivos temporalmente a internet a través de https://0x0.st/

Usage: subir_archivo_temporal [OPTIONS]

Options:
  -t, --tiempo <TIEMPO>  tiempo en horas que el archivo debe estar disponible, por defecto 168 (7 días) [default: 168]
  -n, --nojson           evitar crear el archivo de resumen en json
  -h, --help             Print help
  ``````
